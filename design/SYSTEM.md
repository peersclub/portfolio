# SYSTEM.md

This is the SINGLE source of truth for all design tokens. No other values should exist in the codebase.

## Typography

### Font Families
- **Heading**: `Inter`, sans-serif
- **Body**: `Inter`, sans-serif
- **Mono**: `JetBrains Mono`, monospace
